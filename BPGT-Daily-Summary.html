Make sure your TERMINAL PASSENGER DATA Google Sheet is set to "Anyone with the link can view":

  1. Open the Google Sheet
  2. Click Share (top right)
  3. Under "General access" → change to Anyone with the link
  4. Set role to Viewer
  5. Click Done → then hit Refresh on this dashboard

That's it — no Apps Script changes needed. The dashboard reads directly from Google Sheets.
